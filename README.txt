This function is using for overlapping event in a same time.

This is a nodeJS program so please install nodeJS version >= 12

Go to the function root folder and run:
1. node run.js <json file>
2. the result file will be in the same folder with run.js file

Ex: 
1. node run.js exmaple.json
2. The result file will be exmaple.optimal.json.expected
